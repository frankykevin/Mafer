B

Backbone:
columna vertebral. Infraestructura de la transmisión de datos en Internet

Back-endprocessor:
procesador que se utiliza para determinada función muy especializada, como por ejemplo, administrar una base de datos.

Backup:
copia de seguridad. Se hace para prevenir una posible pérdida de información

Bandeja de entrada:
buzón de entrada.

Banner :
gráfico, generalmente rectangular, que se inserta en una página web. Puede tener carácter publicitario.

Barra de herramientas :
conjunto de íconos que conducen a instrucciones.

Base de datos :
conjunto de datos organizados de modo tal que resulte fácil acceder a ellos,gestionarlos y actualizarlos..
